## 📝 概要

このプロジェクトでは、シンプルなQRコードカードをHTMLとCSSで再現しています。  
gridレイアウト設計の基礎を意識してコーディングしました。

## 🧪 作業プロセス

1. フォルダ構成を決定し、HTMLの基本構造を作成
2. `main`にGridを設定して、中央寄せのベースを構築
3. `section`で背景を全幅、`.inner`でコンテンツを中央配置
4. `grid-template-rows`に名前をつけて構造化したCSSを実験
5. GitHub Pagesで公開
